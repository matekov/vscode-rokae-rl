# ROKAE RL for Visual Studio Code

Syntax highlighting and dark theme for ROKAE RL `.mod` files.

This extension provides a lightweight programming environment for ROKAE xCore Robot Language (RL) inside Visual Studio Code. The goal of the project is to make editing RL source code more comfortable while keeping the appearance familiar to the xCore programming environment.

---

## Features

- Automatic recognition of `.mod` files
- Custom ROKAE RL language definition
- Dark theme only yet
- Syntax highlighting for
  - Keywords
  - Data types
  - Robot commands
  - Built-in functions
  - Constants
  - Numbers
  - Strings
  - Comments
- Automatic bracket matching
- Automatic quote pairing
- Line and block comment support

---

## Installation

### Install from VSIX

1. Open Visual Studio Code.
2. Open the Extensions view (`Ctrl+Shift+X`).
3. Click the three dots (`...`) in the top-right corner.
4. Select **Install from VSIX...**
5. Choose the downloaded `.vsix` package.

---

## Usage

Open any ROKAE source file with the `.mod` extension.

If the theme is not automatically selected:

1. Press `Ctrl+K Ctrl+T`
2. Select **ROKAE Dark**

---

.mod
.MOD
