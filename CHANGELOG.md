## [0.0.1] - 2026-07-02

### Added

- Initial public release
- Automatic recognition of `.mod` files
- Custom ROKAE RL language definition
- ROKAE Dark theme
- Syntax highlighting for comments, strings, numbers, keywords, data types, robot commands, built-in functions and constants
- Bracket matching
- Auto-closing brackets and quotes
- Line and block comment support
